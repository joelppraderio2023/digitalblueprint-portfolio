# DigitalBlueprint — Portfolio Web

Sitio web de portfolio y servicios de **DigitalBlueprint**, desarrollador web freelance especializado en diseño custom, SEO y deploy incluido.

## 🚀 Ver en vivo

> [digitalblueprint.netlify.app](https://digitalblueprint.netlify.app) *(próximamente con dominio propio)*

## 📁 Estructura

```
├── index.html               # Página principal (todo en un archivo)
└── assets/
    └── screenshots/         # Capturas de los proyectos del portfolio
```

## 🛠 Screenshots del portfolio

Para actualizar las capturas de los proyectos:

```bash
node take-screenshots.js
```

*(Requiere Node.js y las dependencias instaladas con `npm install`)*
